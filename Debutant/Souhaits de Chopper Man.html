Souhaits de Chopper Man - Catégorie "Débutant"


Explications

- La première partie de ces nouveaux souhaits de Chopper Man est faite pour les débutants, avec des missions simples à effectuer.

- Elle est très linéaire, avec des souhaits qui se débloquent une fois un palier atteint (ceux qui sont en gras).

- Afin de vous aider à comprendre au mieux comment se divise cette première partie, j'ai divisé en plusieurs tableaux les souhaits débloqués sur chaque palier.

- Vous n'êtes pas obligés de terminer tous les souhaits d'un palier pour débloquer le suivant.

- Il y aura également le souhait "Renforcer 3 fois des personnages" qui sera assez spécial puisqu'il débloquera le souhait permettant d'obtenir Usopp, appelée la partie 2b et qui ne débloquera pas la partie 3.

- Vous devez terminer tous les souhaits de Chopper Man de la catégorie "Débutant" pour débloquer les souhaits de Chopper Man de la catégorie "Intermédiaire".

Souhaits de Chopper Man - Catégorie "Maître"


Explications

- La dernière partie de ces nouveaux souhaits de Chopper Man est faite pour les joueurs les plus expérimentés, avec des missions encore une fois assez simples mais qui vont être très longues à réaliser, surtout sur certains points.

- Contrairement aux deux premières parties, ces souhaits vont se diviser par thème, avec pour chacun le premier souhait disponible dès le déblocage des souhaits de Chopper Man de la catégorie "Maître" puis les suivants qui débloquent dès que le précédent est terminé.

- Par exemple, si vous faites le premier souhait pour les évolutions qui est "Évoluer 2 personnages", vous réussirez ainsi le souhait de Chopper Man et vous débloquerez le souhait suivant qui sera une nouvelle fois "Évoluer 2 personnages".

- Pour les souhaits de Chopper Man concernant le P-LVL, vous devrez finir une quête lors de chaque souhait pour le réussir si jamais vous êtes déjà au-dessus du P-LVL indiqué.

- Pour les souhaits de Chopper Man concernant l'île des collectionneurs, vous aurez juste à la visiter afin de réussir tous les souhaits en même temps si jamais vous avez déjà atteint les niveaux indiqués sur les différentes installations.

- Afin de vous aider à comprendre au mieux comment se divise cette troisième partie, j'ai divisé en plusieurs tableaux les souhaits débloqués sur chaque thème.

- Le souhait de Chopper Man sera terminé dès que vous aurez atteint le P-LVL indiqué.

- Il faudra terminer une quête par souhait, vous ne pourrez tous les débloquer en même temps si vous êtes déjà au-dessus du P-LVL 250.
